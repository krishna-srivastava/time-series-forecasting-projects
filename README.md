# 📈 Time Series Forecasting Projects

End-to-end Time Series ML projects built with Python, covering EDA, Feature Engineering, Forecasting Models and Power BI Dashboards.

---

## 🗂️ Projects

### 📦 Project 1 — Simple Sales Analysis
> Beginner-level time series project to understand core concepts.

- Datetime handling, Resampling, Rolling Window
- Moving Average, Lag Features
- Weekly & Monthly trend analysis

---

### 🏪 Project 2 — Rossmann Store Sales Forecasting
> Retail sales forecasting on real Kaggle dataset (911K rows, 1115 stores).

- **EDA** — Holiday effect, Promo analysis, Store-wise performance
- **Feature Engineering** — Lag features, Rolling mean, Date features
- **Models** — Random Forest, XGBoost, SARIMA
- **Dashboard** — Power BI (Sales trend, Promo impact, Holiday effect)

---

### ⚡ Project 3 — Hourly Electricity Demand Forecasting
> Industry-level forecasting on 3 years of hourly energy consumption data.

- **EDA** — Hourly patterns, Seasonal analysis, Weather impact
- **Feature Engineering** — lag_1, lag_24, lag_168, Rolling features
- **Model** — XGBoost (Next 24 hours forecast)
- **Dashboard** — Power BI (Demand trend, Season-wise, Hourly pattern)
- **Forecasting** — Next day prediction

---

### 📊 Project 4 — Stock Market Analysis (TCS)
> Stock market analytics platform with technical indicators and direction prediction.

- **EDA** — Price trend, Volume analysis, COVID crash analysis
- **Technical Indicators** — RSI, MACD, Bollinger Bands
- **Model** — Random Forest Classifier (UP/DOWN prediction)
- **Forecasting** — Next day direction with probability score
- **Dashboard** — Power BI

> ⚠️ Stock predictions are for educational purposes only. Not financial advice.

---
